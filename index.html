Esta é a declaração do tipo de documento (DOCTYPE). Ela informa ao navegador que este é um documento HTML5.
É a primeira linha em um documento HTML5 e garante que o navegador renderize a página no modo padrão, em vez do modo de compatibilidade com versões antigas do HTML.


lang="pt-BR": Este atributo especifica o idioma principal do documento como português. Isso é importante para acessibilidade, mecanismos de busca e ferramentas de tradução.
Os buscadores usam esses atributos pra entregar páginas da preferencia regional e de idioma de uma pessoa que esta buscando.

O elemento < head > , contém metadados sobre o documento HTML, como o título da página, links para folhas de estilo CSS, scripts JavaScript e outras informações que não são exibidas diretamente na página.


Esta tag < meta> define a codificação de caracteres do documento como UTF-8. UTF-8 é uma codificação de caracteres universal que suporta a maioria dos caracteres usados em diferentes idiomas, garantindo que o texto seja exibido corretamente.


Esta tag <meta> configura a viewport para dispositivos móveis. width=device-width: Define a largura da viewport para a largura do dispositivo. initial-scale=1.0: Define o nível de zoom inicial para 1.0, garantindo que a página seja exibida sem zoom inicial. Essa meta tag é essencial para tornar as páginas da web responsivas e adaptáveis a diferentes tamanhos de tela.
